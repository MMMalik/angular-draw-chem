aim of the project The aim of this project is to create a light-weight molecule editor for drawing structural formulas (organic structures mostly). The editor is based on AngularJS and is from the beginning designed to support svg, enabling drawing of good-looking, scalable structures, ready to embed within html. For example, the structure below (known as Paclitaxel) was drawn using adChem.

how to use it You can use it by adding mmAngularDrawChem module to your project, making it available for the users of your website. More information can be found in the download and manual sections. Also, do not hesitate to use editor directly on this website: make structures that you need, grab the svg and place it within your html!

background There are several molecule editors designed for drawing structural formulas (wiki). Most of them are, however, propietary (non-free) software or are at best licensed under more permissive GPL. adChem is, on the other hand, MIT licensed and it can be easily embedded within any AngularJS app (see download section for details). It is not a big issue if your website is not based on AngularJS - you can still use adChem, since AngularJS can be easily integrated with an existing code already written with other frameworks in mind.

issues If you spot anything wrong or want to comment on any problem, just open an issue. As for the cross-browser compatibility, the app was extensively tested on Chrome (48) and Firefox (44) on Ubuntu 14.04 and no major issues were spotted. A small issue arises on Chrome when using ctrl+t shortcut for transferring svg, since the browser opens a new tab instead and I couldn't override this default behaviour. Under Windows 10, there seems to be a conflict between adChem's onMouseMove events and Firefox's (44) default behaviour ('cloned' structures appearing in lower-right corner, slower reaction to mouse move). No such problems were noticed on Chrome (48) and Edge (25).
