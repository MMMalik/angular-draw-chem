aim of the project The aim of this project is to create a light-weight, AngularJS-based editor for producing structural formulas. The editor is from the beginning designed to be svg-based, thus enabling the drawing of good-looking, scalable structures.

how to use it You can use it by adding mmAngularDrawChem module to your project, making it available for the users of your website, so they can make their own structures. Also, do not hesitate to use it directly on this website: make structures that you need, grab the svg and place it within your html!
